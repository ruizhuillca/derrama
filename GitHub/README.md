# Mapas QGIS

## Capas exportadas

- **Oficina**: 64 elementos (Punto)
- **Zonas**: 280 elementos (Polígono)
- **Territorio**: 320 elementos (Polígono)

## Archivos

- `index.html` - Página web principal
- `data/` - Carpeta con datos GeoJSON
  - `oficina.geojson` - Datos de Oficina
  - `zonas.geojson` - Datos de Zonas
  - `territorio.geojson` - Datos de Territorio

## Para GitHub Pages
1. Sube todos los archivos a GitHub
2. Ve a Settings > Pages
3. Configura: Source: main, Folder: / (root)
